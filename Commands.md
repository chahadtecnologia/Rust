# Commands used by Rust and Cargo
## Command to create a new project using Cargo
> cargo new <project_name>

## Command to compile Rust .rs file
> rustc <file.rs>

## Command to compile a project using Cargo
### Debug Mode
> cargo build

### Relase Mode
> cargo build --release

## Command to build and run the project with a single command
> cargo run 

## Command to check for code errors
> cargo check

## Command to clean all projects
> cargo clean

## Command to format the code
>rustfmt <file.rs>
> cargo fmt
